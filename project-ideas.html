Group Project Idea

It is difficult to create a new project idea and include core aspects of each group members when the scope and goal for each project can be completely different or unrelated. Together we discussed possibilities on how we can amalgamate our ideas into one. After considering various options, we decided on a final idea: a Raspberry Pi powered Portable Game Device.

What exactly is it?

A Raspberry Pi which has local game files stored on a Linux OS, which can be accessed via a browser, allowing a user to play the game(s) available.

How does it work?

Using a modern browser such as Google Chrome, a user can interface and play any game(s) available. The Raspberry Pi stores the game files locally, providing a portable experience. The game files will be a mixture of HTML5, CSS3 and JavaScript files, programmed to create a functioning game.

What else is needed for it to work?

A display device, such as a TV or Monitor is required, along with an HDMI cable. A keyboard and/or mouse is also required to interact with the game(s). A possibility is creating a specially designed controller for the game - similar to old style console game controllers, comprising of a joystick, and some buttons.

How will this be created?

After acquiring a Raspberry Pi model and installing a Linux Distro on a Micro SD card, we will need to design, and program a game to run in a browser. Using a JavaScript framework to assist in programming the game will be most helpful, as there are many frameworks in existence which are designed specifically to create games. Our idea will be to create more than one game if time permits, however we are aware this is a difficult goal to achieve, and will aim to at least have a single functioning game by the end.
